SecureLoc

Should be run with Panda3D 1.10, embedding Python 3.6.

Make sure the python directory of Panda 3D is in your Path.

Simply run with: python main.py.

Useful parameters :

In paramaters.py :

set ENABLE_LOGS to 1 to keep track of the ranging, position and speed measurements in text file. set PLAYBACK to True to replay existing logs rather. set DEBUG to 1 to get more debug informations. Modify the server IP address to match yours if not using the playback mode.

anchors.tab : contains the anchors position. Should the closest possible to the reality.

Running Logs will display the speed data from the available logs (python Logs.py) Modify the main to display whatever data judged useful.

Saturation filter available in Filter.py is not used yet, altough it works. Only the sliding window is used. Modify the main for filters testing (python Filter.py).

Keyboard controls: q to quit w to enable sliding window
